## 新增新的条目

### 支持：

1. 添加`别名`以方便创建近义条目

2. 类别`自动补全`，防止输入不存在的类别

3. 自定义`标签`，优化搜索结果

![](http://olxhi15qe.bkt.clouddn.com/complete.png)

### 支持：

1. Markdown的`富文本编辑器`，实时预览结果，所见即所得

2. 可选自动`扫描`文章中出现的维基系统中已经存在的关键词条目，并将其转为相应的跳转链接

![](http://olxhi15qe.bkt.clouddn.com/editor.png)
